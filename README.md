# Connect-Four AI

Welcome! This is a CLI-based Connect-Four game, featuring an intelligent AI. Utilizing Minimaxing, with alpha-beta pruning, move-ordering, positional weight mapping and more, the AI can easily whoop your ass!

## How to play:
1. Open your terminal and navigate to "src"
2. Compile code with `javav Main.java`
3. Run program `java Main` and enjoy!
